# FoodHub Delivery Analysis

Exploratory analysis and machine learning project on food delivery order data,
using Python and SQL (SQLite via SQLAlchemy) — no SQL workbench required.

## Tech Stack
Python · SQLite · SQLAlchemy · pandas · scikit-learn · seaborn · Jupyter

## Project Structure
data/          # raw CSV (not committed)
database/      # SQL scripts and schema
notebooks/     # Jupyter notebooks
src/           # reusable Python modules
outputs/       # saved charts and figures

## How to Run
pip install -r requirements.txt
python database/ingest.py
